# NaminTax

VAT and tax consulting service website built with React and Vite.

## About

NaminTax is a professional VAT and tax consulting service dedicated to helping businesses navigate the complexities of taxation. This website provides information about our services and allows potential clients to learn more about how we can help them.

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm (v10 or higher)

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Build

Build the application for production:
```bash
npm run build
```

### Preview

Preview the production build:
```bash
npm run preview
```
